# Kseniia — Сайт нутрициолога

## Что это
Лендинг для Ксении Галаниной, нутрициолога. Чистый HTML, хостится на GitHub Pages.

## Структура
```
index.html   — весь сайт в одном файле (контент + стили + скрипты)
CNAME        — домен для GitHub Pages
*.jpg        — фото
```

## Деплой
Push в `main` = автоматическое обновление сайта. Никакого билда, никаких зависимостей.

## Как редактировать
Всё в `index.html`. Найди нужный блок по тексту и правь напрямую.
